# PySide6 迁移规范

## 需求理解

| 目标 | 优先级 | 说明 |
|------|--------|------|
| UI 风格适配 | P0 | Windows 原生风格适配，采用 Fluent Design 视觉语言 |
| 布局能力增强 | P0 | 响应式布局，Flutter 风格的声明式布局 DSL |
| 视觉效果 | P1 | 窗口半透明效果（磨砂玻璃/模糊背景） |
| 系统托盘 | P1 | 程序最小化至系统托盘运行 |
| 兼容性保障 | P0 | 重构过程中确保原有功能完整保留 |

## 实施约束

1. **以给定输入为准**：不做功能扩展推测
2. **布局方案**：在开发过程中逐步明确
3. **渐进式重构**：优先保证功能完整性
4. **平台限制**：本期仅支持 Windows 平台
5. **UI 设计**：参考原 Tkinter 设计，去掉顶部提示区

## UI 布局设计

### 整体结构

窗口采用**双区结构**：**上文字区** + **下按钮区**

### 上文字区布局

上文字区拆分为**左区**和**右区**：

| 区域 | 组成 | 尺寸 | 对齐方式 |
|------|------|------|----------|
| 左区 | 左文字区 + 润按钮 | 左文字区 350x400 | 向上对齐 |
| 右区 | 右文字区 | 600x500 | 向上对齐 |

**布局说明：**
- 左区采用垂直布局（`QVBoxLayout`），左文字区在上，润按钮在下
- 右区采用垂直布局（`QVBoxLayout`），仅包含右文字区
- 左右区之间间距为 10px
- 左区和右区均设置为向上对齐（`Qt.AlignmentFlag.AlignTop`）

### 下按钮区布局

下按钮区包含**复制按钮**和**Accept按钮**：

| 按钮 | 对齐规则 | 说明 |
|------|----------|------|
| 复制按钮 | 与上文字区右区左对齐 | 暂不实现功能 |
| Accept 按钮 | 与上文字区右区右对齐 | 位置不变 |

**布局说明：**
- 左侧留白 350px（匹配左区宽度）
- 按钮布局间距 10px
- 复制按钮起始位置：350 + 10 = 360px（与右区左边界对齐）
- 中间使用弹性空间（`QSizePolicy.Policy.Expanding`）
- Accept 按钮与右区右边界对齐

### 布局示意图

```
+------------------------------------------------------+
|                       上文字区                        |
|                                                      |
|  +----------------+      +----------------------+     |
|  |   左区         |      |       右区            |     |
|  +----------------+      +----------------------+     |
|  | 左文字区       |      |                      |     |
|  |  350x400       |      |    右文字区           |     |
|  |  (向上对齐)    |      |   600x500            |     |
|  +----------------+      |   (向上对齐)          |     |
|  |    润按钮      |      +----------------------+     |
|  +----------------+                                   |
|                                                      |
+------------------------------------------------------+
|                       下按钮区                        |
|                                                      |
|  [留白350px]  [复制按钮]  [弹性空间]  [Accept]       |
|      ↑           ↑                        ↑           |
|   左区宽度    右区左对齐              右区右对齐       |
|              (350+10=360px)                        |
|                                                      |
+------------------------------------------------------+
```

### 对齐计算

**上文字区布局：**
- 左区宽度：350px
- 间距：10px
- 右区起始位置：350 + 10 = 360px

**下按钮区布局：**
- 左侧留白：350px
- 按钮布局间距：10px
- 复制按钮起始位置：350 + 10 = 360px ✅（与右区左边界对齐）

### 实现要点

1. **左区布局**：使用 `QVBoxLayout`，设置 `setAlignment(Qt.AlignmentFlag.AlignTop)`
2. **右区布局**：使用 `QVBoxLayout`，设置 `setAlignment(Qt.AlignmentFlag.AlignTop)`
3. **按钮区对齐**：通过固定宽度的留白控件（`QWidget`）实现精确对齐
4. **响应式考虑**：当前使用固定尺寸，后续可考虑响应式布局
